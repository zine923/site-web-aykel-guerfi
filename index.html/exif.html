Qu'est-ce qui se cache dans vos photos ?

L'EXIF (Exchangeable Image File Format) est une étiquette numérique collée à chaque image JPEG.

Informations stockées :

  • Géolocalisation : Les coordonnées GPS exactes du lieu de la photo.
  • Paramètres techniques : Vitesse d'obturation, ouverture du diaphragme, ISO.
  • Appareil : Marque, modèle et version du logiciel.
