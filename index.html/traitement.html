Des électrons aux pixels

Le signal électrique sortant du capteur est transformé en nombres. Chaque pixel possède une valeur pour le Rouge, le Vert et le Bleu (système RVB).

Ce qu'on fait sur une image :

  • La Définition : C'est le nombre total de pixels (L x H).
  • Le Codage : Chaque couleur primaire est souvent codée sur 8 bits (valeurs de 0 à 255).
  • Le Format JPEG : Une compression qui réduit la taille du fichier en perdant un peu de qualité.
  • Le Format RAW : Le fichier brut du capteur, sans aucune modification.
